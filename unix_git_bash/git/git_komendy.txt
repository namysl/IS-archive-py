git remote -> jakie zdalne galezie zostaly podlaczone
git remote -v

git remote add [skrot] [url]

git fetch [nazwa zdalnego repo] -> sciaga nowe zmiany, ktorych
nie masz ze zdalnego repo
nie merge'uje twoich danych ze zdalnym repo

git pull -> sciaga i merge'uje

git push [nazwa zdalnego repo] [nazwa galezi] -> wypchanie zmian
musisz miec prawo zapisu, musisz miec aktualne dane z repo
(jesli ktos w miedzyczasie wypchnal zmiany, musisz je najpierw 
pobrac i scalic)

git remote show [nazwa zdalnego repo] -> inspekcja zdalnych zmian

git remote rename [stara] [nowa] -> zmiana nazwy odnosnika

git remote rm [nazwa] -> usuwanie
